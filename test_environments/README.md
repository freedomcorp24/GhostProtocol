# GhostProtocol Test Environments

This directory contains test environments for GhostProtocol web, iOS, and Android functionality.

## Directory Structure

- `web/`: Web test environment
- `ios/`: iOS test environment
- `android/`: Android test environment
- `server/`: Server configuration and scripts

## Starting the Test Environments

To start the test environments, run the following command:

```bash
./start_test_environments.sh
```

This will start the following servers:

- Web: http://localhost:8080
- iOS: http://localhost:8081
- Android: http://localhost:8082

## Accessing the Test Environments

The test environments can be accessed using the following URLs:

- Web: http://localhost:8080
- iOS: http://localhost:8081
- Android: http://localhost:8082

## Test Environment Features

Each test environment demonstrates the following GhostProtocol components:

- Authentication System
- Secure Storage (Vault)
- Cryptocurrency Integration
- Premium Tiers and Subscription System
- Admin Panel

## Notes

These test environments are for demonstration purposes only and do not include actual functionality. They are meant to show how the GhostProtocol components would be implemented using Signal's native implementation.
