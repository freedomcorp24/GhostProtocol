When testing the development environment, please verify the following functionality:

1. User authentication
2. Message sending/receiving

4. File sharing

## Troubleshooting

If you encounter any issues accessing the URLs, please check the following:

1. Ensure your network allows access to the IP address
2. Verify the URLs are entered correctly
3. Check that the services are running on the EC2 instance

For any persistent issues, please contact the development team.
