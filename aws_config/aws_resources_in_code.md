# AWS Resources Referenced in GhostProtocol Code

## S3 Buckets
- `ghostprotocol-dev-profiles`: User profile data and images
- `ghostprotocol-dev-vault`: Secure storage for user vault items
- `ghostprotocol-dev-media`: Media files shared in messages
- `ghostprotocol-dev-attachments`: File attachments for messages

## DynamoDB Tables
- `ghostprotocol-dev-accounts`: User account information
- `ghostprotocol-dev-devices`: User device information
- `ghostprotocol-dev-messages`: Message storage
- `ghostprotocol-dev-groups`: Group chat information
- `ghostprotocol-dev-crypto-wallets`: Cryptocurrency wallet information
- `ghostprotocol-dev-storage-usage`: User storage usage tracking
- `ghostprotocol-dev-admin-roles`: Admin role assignments
- `ghostprotocol-dev-vault-items`: Secure vault items
- `ghostprotocol-dev-subscriptions`: User subscription information

## AWS Region
- Primary region: `us-west-1` (US West - N. California)
